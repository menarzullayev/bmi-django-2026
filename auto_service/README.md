# АвтоСервис ERP

Дипломная работа: система управления заказами и клиентской базой автосервиса.

## Описание

ERP-система для автосервиса с разделением ролей (администратор, приёмщик, механик),
учётом клиентов, автомобилей, заказ-нарядов, сервисных работ, складских запчастей,
канбан-доски статусов и выставления счетов с печатью PDF.

## Технологии

- Python 3.12, Django 5.1
- SQLite, Pillow, ReportLab
- TailwindCSS (CDN), HTMX 2.0, Lucide иконки, Chart.js
- Шрифт DejaVuSans для кириллицы в PDF

## Установка

```bash
cd /home/nsn/Django_projects/auto_service
python3 -m venv venv
source venv/bin/activate
pip install django pillow reportlab
python manage.py migrate
python manage.py seed_data
```

## Запуск

```bash
source venv/bin/activate
python manage.py runserver 8766
```

Открыть в браузере: <http://127.0.0.1:8766/>

## HTMX-функции

1. Канбан-доска: кнопка «Далее» переводит заказ на следующий этап без перезагрузки.
2. Корректировка остатков склада (+1/−1) в таблице и карточке запчасти.
3. Добавление сервисной работы в заказ-наряд без перезагрузки.
